# 숫자 맞추기 게임 🎯

## 목표

숫자 맞추기 게임은 컴퓨터가 무작위로 생성한 숫자를 사용자가 맞추는 간단하면서도 재미있는 게임입니다.

---

## 게임 규칙 🕹️

1. 컴퓨터는 **1부터 100 사이의 무작위 숫자**를 생성합니다.
2. 사용자는 숫자를 입력하며, 컴퓨터로부터 **힌트(높다/낮다)**를 받습니다.
3. 정답을 맞출 때까지 반복합니다.
4. 정답을 맞추면 **축하 메시지**가 출력됩니다.

---

## 사용 기술 🛠️

- **HTML**: UI 구조를 작성합니다.
- **CSS**: 스타일링 및 사용자 경험을 개선합니다.
- **JavaScript**: 게임 로직을 구현합니다.

---

## 주요 기능 📌

1. **난수 생성 및 게임 시작**

   - 게임 시작 버튼을 누르면 **1부터 100 사이의 난수**가 생성됩니다.
   - 시도 횟수와 입력 기록이 초기화됩니다.

2. **숫자 입력 및 비교**

   - 사용자가 입력한 숫자가 타겟보다 높은지 낮은지 비교하여 메시지로 표시합니다.
   - 정답을 맞출 때까지 반복하며, 정답 시 **축하 메시지**를 표시합니다.

3. **시도 횟수 및 입력 기록**

   - 입력할 때마다 시도 횟수가 **실시간으로 업데이트**됩니다.
   - 지금까지 입력한 숫자가 **기록**되어 화면에 표시됩니다.

4. **입력 필드 관리**
   - 입력 후 **숫자가 자동으로 초기화**됩니다.
   - **Enter 키**를 눌러 입력 버튼을 클릭하는 기능도 제공됩니다.

---

## 추가된 내용 ✅ 24.11.30

1. **현재 입력한 숫자 표시**

   - 입력한 숫자와 관련된 **힌트 메시지**를 제공합니다. (예: "50보다 더 높습니다.")

2. **Enter 키 입력 가능**

   - **Enter 키**를 눌러 입력 버튼과 동일한 기능을 수행하도록 개선했습니다.

3. **입력 후 숫자 초기화**

   - 숫자 입력 후 **입력란이 자동으로 비워집니다**.

4. **입력한 숫자 기록 표시**

   - 사용자가 입력한 모든 숫자가 화면에 **기록**되어 가시성을 높입니다.

5. **시도 횟수 표시**
   - 입력할 때마다 **시도 횟수가 증가**하며 화면에 **실시간으로 표시**됩니다.
