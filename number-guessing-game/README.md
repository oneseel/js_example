# Number Guessing Game

## 🎯 목표

숫자 맞추기 게임은 컴퓨터가 무작위로 생성한 숫자를 사용자가 맞추는 간단한 게임입니다.

---

## 🕹️ 게임 규칙

1. 컴퓨터는 1부터 100 사이의 무작위 숫자를 생성합니다.
2. 사용자는 숫자를 입력하고 컴퓨터의 힌트(높다/낮다)를 받습니다.
3. 정답을 맞출 때까지 시도합니다.
4. 정답을 맞추면 축하 메시지가 출력됩니다.

---

## 🛠️ 사용 기술

- **HTML**: UI 구조를 작성.
- **CSS**: 스타일링 및 사용자 경험 개선.
- **JavaScript**: 게임 로직 구현.
